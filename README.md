An interactive AI-powered learning mentor built using Streamlit, LangChain, and Google Gemini API.
This app acts as a domain-specific mentor that answers questions only from the selected learning module, providing structured and educational responses.

Features
Module-based mentoring (Python, SQL, ML, DL, GenAI, Agentic AI, etc.)
Experience-based mentoring (select mentor experience in years)
Strict domain control (answers only related to selected module)
Chat-style interactive UI using Streamlit
Download chat history as TXT or PDF
Powered by Google Gemini (LLM) via LangChain

How It Works
User selects a learning module
User chooses mentor experience (years)
AI responds only to module-related questions
Irrelevant questions are automatically rejected
Chat history can be downloaded as TXT or PDF

Example Use Cases
AI mentor for students & beginners
Interview preparation assistant
Domain-restricted AI chatbot
Learning companion for Data Science & AI topics
